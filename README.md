### Django Control Room
Create a command center out of your Django admin.

* Official website: [Django Control Room](https://djangocontrolroom.com)

| Repo | Description | Stars |
| --- | --- | --- |
| [dj-control-room](https://github.com/yassi/dj-control-room) | Build a control center in your admin with different tools and panels. The missing plugin system for the Django Admin. | [![GitHub Repo stars](https://img.shields.io/github/stars/yassi/dj-control-room?style=social)](https://github.com/yassi/dj-control-room/stargazers) |

#### Panels Available
| Panel | What it does |
| --- | --- |
| [dj-redis-panel](https://github.com/yassi/dj-redis-panel) | Redis visibility inside Django admin |
| [dj-cache-panel](https://github.com/yassi/dj-cache-panel) | Inspect and debug your cache layer |
| [dj-celery-panel](https://github.com/yassi/dj-celery-panel) | Monitor workers and tasks from admin |
| [dj-urls-panel](https://github.com/yassi/dj-urls-panel) | Visualize and test Django URL routing inside the Django Admin |
| [dj-signals-panel](https://github.com/yassi/dj-signals-panel) | See all your registered signals and recievers in the Django admin. |

More coming soon.
