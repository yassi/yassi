### Django Control Room
Create a command center out of your Django admin with a growing suite of open-source tools that live inside the Django admin

* Official website: [Django Control Room](https://djangocontrolroom.com)
* Repo: [dj-control-room](https://github.com/yassi/dj-control-room): Build a control center in your admin with different tools and panels. The missing plugin systetm for the Django Admin.

#### Panels Available
- [dj-redis-panel](https://github.com/yassi/dj-redis-panel): Redis visibility inside Django admin
- [dj-cache-panel](https://github.com/yassi/dj-cache-panel): Inspect and debug your cache layer
- [dj-celery-panel](https://github.com/yassi/dj-celery-panel): Monitor workers and tasks from admin
- [dj-urls-panel](https://github.com/yassi/dj-urls-panel): Visualize and test Django URL routing inside the Django Admin
- More coming soon!

Want to build your own panels and software in the Django Admin? Django Control Room is extensible.
Just use our cookiecutter template to get started:

- [cookiecutter-dj-control-room-plugin](https://github.com/yassi/cookiecutter-dj-control-room-plugin)
